## **Briefing**
A set of projects for stm32f4 microcontrollers.
The purpose of each one is mostly educational.
* _IDE used_: STM32CubeIDE
* _Microcontroller used_: STM32F401VCT6U
* Board used: STM32F401C-DISCO

### **List of available projects:**
* **_led_blink_** - Simple LED blinking program.
* **_cdc_vcp_test_** - USB Communication Device Class (CDC) program that acts like the Virtual COM port under OS.
